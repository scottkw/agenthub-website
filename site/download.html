AgentHub for macOS

Universal binary (Apple Silicon + Intel). Signed with a Developer ID and notarized by Apple.

Direct download — .dmg
agenthub-v3.3-darwin-universal.dmg · 15 MB
Download .dmg

Or via Homebrew

$brew tap scottkw/agenthub && brew install --cask agenthub

Requirements

  • macOS 11 Big Sur or later
  • For the Xcode Command Line Tools: xcode-select --install (only needed if building from source)
  • Optional: Tailscale enables zero-config browser-trusted TLS for shared sessions

AgentHub for Linux

Built for amd64 on Ubuntu 22.04 and 24.04. Both WebKitGTK 4.0 and 4.1 are supported.

Install the .deb

$sudo dpkg -i agenthub-v3.3-linux-amd64.deb

Runtime requirements

  • Ubuntu 22.04+ or Debian 12+ (glibc 2.35+)
  • Ubuntu 24.04 / Debian 13+: libgtk-3 libwebkit2gtk-4.1
  • Ubuntu 22.04: libgtk-3 libwebkit2gtk-4.0
  • D-Bus (for system tray via StatusNotifierItem — GNOME/KDE/XFCE)
